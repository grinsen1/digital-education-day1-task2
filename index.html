Диджитал-Медиапланирование

Добро пожаловать в курс по диджитал-медиапланированию

Научитесь эффективно планировать рекламные кампании в цифровой среде. Изучите основные понятия, площадки и инструменты медиапланирования, а также выполните практические задания для закрепления навыков.

Структура обучения

1

Основы и понятия

Изучите ключевые термины и метрики медиапланирования, такие как CPM, CTR, CPC и другие.

2

Практические задания

Выполните задания по выбору площадок для достижения различных маркетинговых целей.

3

База знаний

Изучите детальную информацию о рекламных площадках и их показателях эффективности.

Обучение медиапланированию

Основные понятия и метрики

CPM (Cost Per Mille)

Стоимость тысячи показов рекламы. Основная метрика для закупки рекламы.

Бенчмарк: руб.

CTR (Click-Through Rate)

Показатель кликабельности рекламы. Отношение числа кликов к показам.

Бенчмарк: %

CPC (Cost Per Click)

Стоимость одного клика по рекламе.

Бенчмарк: руб.

CPV (Cost Per View)

Стоимость одного просмотра видео-рекламы.

Относительный показатель, зависит от CPM и VTR.

VTR (View-Through Rate)

Показатель досмотров видео-рекламы до конца.

Бенчмарк: %

CPT (Cost Per Thousand)

Стоимость тысячи контактов с учетом частоты.

Бенчмарк: руб.

CPA (Cost Per Action)

Стоимость целевого действия на сайте.

Бенчмарк: руб.

PI (Page Interest)

Индекс заинтересованности страницей. Оценивает качество трафика.

Чем выше значение, тем выше качество целевого трафика.

Рекомендации по выбору площадок

Видео-реклама

Преимущества: Высокая вовлеченность, эмоциональное воздействие

Когда использовать: Для повышения узнаваемости бренда, запуска новых продуктов

Ключевые метрики: VTR, CPV, охват, частота

Баннерная реклама

Преимущества: Широкий охват, низкая стоимость

Когда использовать: Для охватных кампаний, ремаркетинга

Ключевые метрики: CPM, CTR, CPC, частота

Социальные сети

Преимущества: Точный таргетинг, высокая вовлеченность

Когда использовать: Для привлечения аудитории с определенными интересами

Ключевые метрики: CPM, CTR, CPC, CPA, охват

Лучшие практики:

  • Для максимизации охвата выбирайте площадки с низким CPM и высокой аудиторией
  • Для качественного трафика на сайт выбирайте площадки с высоким CTR и низким CPC
  • Для Brandformance (охват + конверсия) используйте комбинацию видео и баннерной рекламы
  • Всегда сравнивайте показатели площадок с бенчмарками для принятия решений
